# EasyEmoteBlocker
Chrome Extension "Easy Emote Blocker", hides given emote images.
To install: 
1. Go to chrome://extensions
2. Turn on "Developer mode"
3. Extract EasyEmoteBlocker to a folder
4. Load the unpacked folder in chrome://extensions

Extension GUI

## content.js
